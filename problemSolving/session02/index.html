Variables Are Like Boxes! üì¶

A variable is a name that stores something‚Äîjust like a labeled cardboard box! To use a variable, we follow 3 rules:

1. Name It

Give your box a clear label without spaces, like player_name.

2. Store It

Put data inside using the equals sign: name = "Omar".

3. Use It

Call the box whenever you need it: print(name).

Meet the Data Types!

Strings (Text)

Words and letters. They always need quotes!

name = "Sara"

Integers / Floats

Numbers and decimals. No quotes needed!

age = 12
score = 99.5

Booleans

Like a light switch! It's either True or False.

is_winner = True

Extra Task: Be a Bug Hunter! üêõ

Our robot friend messed up his variables! Can you find the bugs and fix them? Remember: Python is strict about names and commas!

Bug #1

name = "Ahmed"
print(Name)

The Fix:
Python is case-sensitive! Name (with a capital N) is completely different from name. It should be:

print(name)

Bug #2

score = 100
print("Your score is: " score)

The Fix:
Missing a comma! You need a comma to join text and a variable. It should be:

print("Your score is:", score)
