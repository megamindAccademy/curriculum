What Is a Dictionary? üìñ

In real life, a dictionary tells you the meaning of a word.

In Python, a dictionary stores data in pairs like:

student = { "name": "Ali", "age": 12, "grade": "6th" }
üîë

We call "name", "age", and "grade" the keys.
Their values "Ali", 12, and "6th" are the values.

Personal Dashboard (Turtle Graphics) üê¢

Scenario: You are creating a personal dashboard for a user. This dashboard will show: Name, Age, Favorite Color, Score, Favorite Animal. You'll use this data stored in a dictionary and visually display it using turtle.

dashboard.py

#--Personal Dashboard--

import turtle


# Step 1: Collect user data

dashboard = {}

dashboard["name"] = input("Enter your name: ")

dashboard["age"] = input("Enter your age: ")

dashboard["favorite_color"] = input("Enter your favorite color: ")

dashboard["score"] = input("Enter your score: ")

dashboard["animal"] = input("Enter your favorite animal: ")


# Step 2: Draw the dashboard

screen = turtle.Screen()

screen.bgcolor("white")

pen = turtle.Turtle()

pen.penup()

pen.goto(-150, 100)

pen.pendown()

pen.pensize(3)

pen.color("black")


for _ in range(2):

pen.forward(300)

pen.right(90)

pen.forward(200)

pen.right(90)


# Step 3: Write the dictionary info on the screen

pen.penup()

pen.goto(-140, 60)

pen.color("blue")

pen.write(f"Name: {dashboard['name']}", font=("Arial", 14, "bold"))

pen.goto(-140, 30)

pen.write(f"Age: {dashboard['age']}", font=("Arial", 14))

pen.goto(-140, 0)

pen.write(f"Favorite Color: {dashboard['favorite_color']}", font=("Arial", 14))

pen.goto(-140, -30)

pen.write(f"Score: {dashboard['score']}", font=("Arial", 14))

pen.goto(-140, -60)

pen.write(f"Animal: {dashboard['animal']}", font=("Arial", 14))

pen.hideturtle()

turtle.done()
