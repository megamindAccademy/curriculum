The Code: Step-by-Step üíª

1 Ask for how many students

First, we need to know the total number of students to prepare our empty list.

num_students = int(input("Enter number of students: "))

students = []

2 Get and clean each name

  • It loops num_students times and asks for each name.
  • .strip() removes extra spaces.
  • .capitalize() formats the name nicely.

for i in range(num_students):

name = input(f"Enter name of student {i+1}: ").strip().capitalize()

students.append(name)

3 Display the Student List

This shows the full list of student names in a clean, bullet-point format.

print("\nClass List:")

for student in students:
