What is Recursion? ü™û

Imagine you are standing in front of two mirrors facing each other. You see yourself, and inside that reflection, you see yourself again, and again, and again!

In Python, recursion happens when a function calls itself to solve a smaller piece of a big problem.

‚ö†Ô∏è WARNING: Every recursive function needs a base case to know when to STOP. Without it, the function will run forever!

Countdown Timer üöÄ

Scenario: You are launching a rocket! Create a countdown timer that prints numbers from N down to 1, then prints "Blastoff! üöÄ"

countdown.py

def countdown(n):

if n <= 0: # Base Case

print("Blastoff! üöÄ")

else:

print(n)

countdown(n - 1)


countdown(5)

Math Factorial (!) üßÆ

Scenario: In math, the factorial of a number (like 5!) means multiplying it by every number below it:
5! = 5 * 4 * 3 * 2 * 1 = 120
Recursion is PERFECT for this!

factorial.py

def factorial(n):

if n == 1: # Base Case

return 1

else:

# Function calls itself!

return n * factorial(n - 1)


print(factorial(5))

How it works:

  • ‚ñ∂Ô∏è factorial(5) = 5 * factorial(4)
  • ‚ñ∂Ô∏è factorial(4) = 4 * factorial(3)
  • ‚ñ∂Ô∏è factorial(3) = 3 * factorial(2)
  • ‚ñ∂Ô∏è factorial(2) = 2 * factorial(1)
  • üõë factorial(1) = 1 (Base Case!)

Mini Turtle Spiral üê¢üåÄ

Scenario: Let's use recursion to draw shapes! We tell the turtle to draw a line, turn, and then call itself to draw a slightly shorter line!

turtle_spiral.py

import turtle

t = turtle.Turtle()


def draw_spiral(length):

if length < 5: # Base Case

return

t.forward(length)

t.right(90)

# Call itself with a smaller length!

draw_spiral(length - 5)


draw_spiral(100)

turtle.done()

Robot Cleaner ü§ñ

Scenario: Checking rooms... A robot moves from room 1 to room 5 and cleans any dirty room using recursion.

robot_cleaner.py

#--Robot cleaner--

rooms = ["dirty", "clean", "dirty", "dirty", "clean"]


def clean_room(index):

if index >= len(rooms):

print(" All rooms checked.")

return


print(f" Checking room {index + 1}...")


if rooms[index] == "dirty":

print(" Room is dirty. Cleaning now...")

rooms[index] = "clean"

else:

print(" Room is already clean.")


# Move to the next room

clean_room(index + 1)


clean_room(0)
