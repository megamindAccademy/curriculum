What Is a Flowchart? üîÄ

It is a visual representation of the steps of a process in a clear, organized manner.

It's a map of your logic. Programmers use it to plan before writing code.

Name Shape Symbol Usage
Start/Stop
Oval
 Used only at the beginning and end of the flowchart.
Process
Rectangle
 Used to 'do something' for example, assign a variable a value, or carry out a calculation.
Decision
Diamond
 Used to change the direction of a program. This is the only block that can have more than 1 arrow coming out of it, and any arrows should be labelled with possible answers to the question posed.
Input/ Output
Parallelogram
 Used for a user input or to output or print something.
Connector
O
 If there is not enough space to fit the flowchart on one page, connectors can be used to show where the flowcharts should be joined together.
Direction of flow Arrows show how to travel through the flowchart.

  • 5. End

Project: Traffic Light Simulation üö¶

Scenario: Set up your Traffic Lights Screen. Ask the user and the screen is still open to make sure that the color is correct.

traffic_simulation.py

import turtle


# Setup screen

screen = turtle.Screen()

screen.title("Traffic Light Simulation")


# Create turtle for drawing traffic light box

box = turtle.Turtle()

box.penup()

box.goto(-50, -100)

box.pendown()

box.fillcolor("gray")

box.begin_fill()

for _ in range(2):

box.forward(100)

box.left(90)

box.forward(200)

box.left(90)

box.end_fill()

box.hideturtle()


# Create turtles for lights

red_light = turtle.Turtle()

yellow_light = turtle.Turtle()

green_light = turtle.Turtle()


for light in [red_light, yellow_light, green_light]:

light.shape("circle")

light.penup()


# Position lights

red_light.goto(0, 80)

yellow_light.goto(0, 30)

green_light.goto(0, -20)


# Function to turn on a light

def turn_on_light(color):

red_light.fillcolor("gray")

yellow_light.fillcolor("gray")

green_light.fillcolor("gray")


if color == "red":

red_light.fillcolor("red")

print("STOP!")

elif color == "yellow":

yellow_light.fillcolor("yellow")

print("WAIT!")

elif color == "green":

green_light.fillcolor("green")

print("GO!")

else:

print("Invalid input. Please enter red, yellow, or green.")


# Ask the user

user_input = screen.textinput("Traffic Light", "Enter the traffic light color (red, yellow, green): ")

turn_on_light(user_input)


# Keep window open

screen.mainloop()
