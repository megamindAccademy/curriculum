1. Introduction to Python & Variables

Python is a simple programming language that reads a lot like English. We use the print() function to show text on the screen, and we use Variables as storage boxes for our data.

💡 Variable Naming Rules: Must start with a letter. You can use Camel case or Pascal case. They are case-sensitive (`Name` is different from `name`). Don't use reserved keywords!
intro.py
# Printing a message
print("Hello, Megaminds!")

# Variables and Concatenation
name = "Hassan"
age = "12"
print("Welcome " + name + ", age: " + age)
Hello, Megaminds!
Welcome Hassan, age: 12

2. Data Types and Lists

There are several data types: Text (str), Numbers (int, float), and Booleans (bool). We can use the type() function to find out the data type, and the input() function to get data from the user.

Lists: We use square brackets [] to create a list. We can modify them, add items using append(), or order them using sort().

lists.py
# Data Types
number = 10
print(type(number))

# Lists operations
my_list = ["Ali", "Ahmed"]
my_list.append("Laila") # Add to end
print(my_list)
<class 'int'>
['Ali', 'Ahmed', 'Laila']

3. Math Operations & String Functions

In Python, we can use Arithmetic operations (+ - * / % ** //), Comparison operations (== != > <), and Logical operations (and or not).

Also, Strings are treated as lists of characters! They have special functions like upper() to make them uppercase, and len() to get their length.

strings_math.py
# Math and Logic
x = 5
y = 2
print(x ** y) # 5 to the power of 2

# String Functions
sentence = "welcome"
print(sentence.upper())
print(len(sentence))

4. Tuples and Dictionaries

Tuples: They are just like lists, but we use parentheses () and they are unchangeable after they are created.

Dictionaries: Used to store data values in (Key: Value) pairs using curly brackets {}.

data_structures.py
# Tuple (Cannot be changed)
my_tuple = ("apple", "banana")
print(my_tuple[0])

# Dictionary (Key: Value)
car_dict = {
    "brand": "Ford",
    "year": 1964
}
car_dict["year"] = 2020 # Update value
print(car_dict["brand"])

5. Conditions (If) & While Loops

We use if, elif, else to make decisions in our code. We use a while loop to repeat a block of code as long as a condition is true.

⚠️ Always remember to add code to increment the variable (e.g., i += 1) inside a while loop so it doesn't continue forever (Infinite Loop)!
logic_loops.py
# If Condition
age = 15
if age >= 18:
    print("Adult")
else:
    print("Child")

# While Loop
i = 1
while i < 4:
    print(i)
    if i == 2:
        break # Stop the loop
    i += 1

6. For Loops & Nested Loops

A for loop is used for iterating over a sequence (like a list or a string). We use the range() function to loop through a block of code a specified number of times.

Nested Loops: A loop inside a loop! The "inner loop" will be executed completely for every single iteration of the "outer loop".

for_loops.py
# For loop with range
for x in range(3):
    print(x)

# Nested Loops
adj = ["red", "big"]
fruits = ["apple", "cherry"]
for a in adj:
    for f in fruits:
        print(a, f)
