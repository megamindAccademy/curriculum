The Code: Step-by-Step üíª

1 The Attendance Dictionary 

students = ["Omar", "Menna", "Ali", "Sara"]
attendance = {}

for name in students:
    status = input(f"Is {name} Present (P) or Absent (A)? ").upper()
    attendance[name] = "Present" if status == "P" else "Absent"

We use a dictionary attendance = {} where the keys are student names, and the values are 'Present' or 'Absent'.

2 Summarizing the Data 

present_count = list(attendance.values()).count("Present")
absent_count = list(attendance.values()).count("Absent")

print(f"\nTotal Present: {present_count}")
print(f"Total Absent: {absent_count}")

.values() gets all the statuses. list(...) turns them into a list. .count('Present') counts how many times 'Present' appears.

3 Saving to a File 

import datetime

save = input("Save to file? (yes/no): ").lower()
if save == "yes":
    with open(f"attendance_{datetime.date.today()}.txt", "w") as file:
        file.write("Attendance Report\n")
        file.write("=================\n")
        for name in students:
            file.write(f"{name}: {attendance[name]}\n")
        file.write(f"\nDate: {datetime.date.today()}")
    print(" File saved successfully.")
  • The program asks if you want to save the results. If yes, it creates a .txt file named with today's date and writes the report.
  • with open(..., "w") ensures the file is properly closed after writing.
