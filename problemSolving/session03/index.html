Review & Warm-Up: Do you Remember Logic gates?

Computers make decisions just like us - by asking YES or NO questions.

They use Logic Gates to decide what to do based on answers.

Think of them like tiny brain switches that say "Yes" or "No" X!

Logic Gate Real-World Example Meaning
AND Must have both key AND password to open a door. Both conditions must be true
OR Win a prize if you score high OR find the hidden treasure. At least one condition must be true.
NOT If NOT raining, go outside. Reverses the condition.

Traveling System ‚úàÔ∏è

Scenario: A travel system checks if you are eligible to travel abroad, based on whether you have a valid passport, visa, and whether you have received all vaccinations required by the destination country.

Discussion:

  • 1. AND: All 3 conditions must be true to allow travel.
  • 2. If any condition fails (e.g., missing visa or passport), the traveler cannot proceed.

# Travel

passport_valid = input("Do you have a valid passport? (yes/no): ")

visa_valid = input("Do you have a valid visa for the destination? (yes/no): ")

vaccinations_complete = input("Have you received all required vaccinations? (yes/no): ")

if passport_valid == "yes" and visa_valid == "yes" and vaccinations_complete == "yes":

print("You're good to go! Have a great trip! ")

else:

print("Please ensure all requirements are met before traveling.")

Shopping Cart Validation üõí

Scenario: Before proceeding to checkout in an online store, the website must check that:

Discussion:

  • 1. The cart is not empty.
  • 2. The user has a valid coupon.
  • 3. The total amount is above a minimum for free shipping.

#Shopping cart validation

cart_items = int(input("How many items are in your cart? "))

coupon_valid = input("Do you have a valid coupon? (yes/no): ")

total_amount = float(input("Enter the total amount of your cart in EGP: "))

if cart_items > 0 and coupon_valid == "yes" and total_amount >= 50:

print("Proceed to checkout. Free shipping applied!")

elif cart_items == 0:

print("Your cart is empty. Please add some items first.")

elif coupon_valid == "no":

print("Please apply a valid coupon to get discounts.")

elif total_amount < 50:

print("Add more items to get free shipping.")

else:

print("There's an issue with your cart. Please check.")
