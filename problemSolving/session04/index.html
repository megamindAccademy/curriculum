What is a Loop? üé¢

Do you remember?

A loop is a set of instructions that a computer repeats to complete a task!

Why Loops? Let's Discuss!

A loop helps us repeat a task without rewriting code.

"Wake up ‚û°Ô∏è Get dressed ‚û°Ô∏è Go to school ‚û°Ô∏è Repeat every day"

Computers love loops because they are fast and efficient!

What if We Didn't Use Loops? ‚ùå

  • Would you like to write print('hi') 100 times?
  • Loops help us avoid repeating ourselves
  • Saves time + less mistakes

Types of Loops in Python üêç

For Loop

  • repeat a set number of times
  • You know how many times to repeat
Syntax of a for Loop: for item in list:
    # Do something with item

While Loop

  • repeat while something is true
  • Repeat until a condition is no longer true
Intro to while Loops: i = 0
while i < 5:
    print(i)
    i += 1
