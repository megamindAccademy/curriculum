Quick Revision: What Are Lists?

  • Lists store multiple values in one variable
  • Can hold numbers, strings, or even other lists (nested lists)
  • We access elements by index (starting from 0)
  • Common operations: add, remove, loop over elements

Shopping Cart Total with Discounts üõí

Scenario: You bought several items. If the total cost is over $100, you get a 10% discount!

shopping_cart.py

#--Shopping Cart Total with Discounts--

items = ["shoes", "shirt", "hat", "jacket"]

prices = [45, 25, 15, 60]

total = 0


for price in prices:

total += price


print(f"Total before discount: ${total}")

if total > 100:

discount = total * 0.1

total -= discount

print(f"Discount applied: ${discount}")

else:

print("No discount applied.")

print(f"Total to pay: ${total}")

Discussion:

  • ‚ùì How does the loop calculate total?
  • ‚ùì What happens when total is exactly $100?

Student Grades Tracker üéì

Scenario: You have 3 students with scores for 3 exams, Calculate average and check if each passed (>60%).

grades.py

#--Student Grades Tracker--

students = ["Alice", "Bob", "Charlie"]

scores = [

[75, 80, 68], # Alice's scores

[55, 60, 58], # Bob's scores

[90, 92, 88] # Charlie's scores

]


for i in range(len(students)):

avg = sum(scores[i]) / len(scores[i])

status = "Passed" if avg >= 60 else "Failed"

print(f"{students[i]}: Average {avg:.2f} {status}")

Discussion:

  • ‚ùì How do nested lists store multiple scores?
  • ‚ùì Why do we divide by the length of scores?
  • ‚ùì What if some students had more exams than others?

Password Validator üîí

Scenario: Check if a password is strong based on length, numbers, letters, and spaces.

password_check.py

#--password validation--

password = input("Enter your password: ")


if len(password) < 8:

print("Password too short.")

elif " " in password:

print("Password should not contain spaces.")

elif not any(char.isdigit() for char in password):

print("Password should contain at least one number.")

elif not any(char.isalpha() for char in password):

print("Password should contain letters.")

else:

print("Password is valid!")

Discussion:

  • ‚ùì How do any() and for loop work together?
  • ‚ùì Why check spaces separately?
  • ‚ùì What other password rules could be added?

Grocery List / Shopping List üßæ

Scenario: Create a list of your favorite foods and prices. Add a function that adds a new food & price, and another that calculates total cost.

grocery_list.py

foods = []

prices = []


def add_item():

food = input("Enter food name: ")

price = float(input("Enter food price: "))

foods.append(food)

prices.append(price)


def show_total():

total = 0

for price in prices:

total += price

print("\nShopping Summary:")

for i in range(len(foods)):

print(f"{foods[i]} - ${prices[i]:.2f}")

print(f"\nTotal Cost: ${total:.2f}")


# Main Loop

while True:

choice = input("Do you want to add a food item? (Y/N): ").strip().upper()

if choice == "Y":

add_item()

else:

break

show_total()

Turtle Race Simulation üê¢üèÅ

Scenario: Three turtles race different random distances. Who wins?

turtle_race.py

import turtle

import random


screen = turtle.Screen()

colors = ["red", "blue", "green"]

distances = [random.randint(50, 150) for _ in range(3)]

turtles = []


for i in range(3):

t = turtle.Turtle()

t.color(colors[i])

t.penup()

t.goto(-150, 100 - i*50)

t.pendown()

turtles.append(t)


for i in range(3):

turtles[i].forward(distances[i])


winner_index = distances.index(max(distances))

print(f"The winner is the {colors[winner_index]} turtle!")

turtle.done()
